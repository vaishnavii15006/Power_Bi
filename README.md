Sales Forecasting & Anomaly Detection using Power BI

 Project Overview

This project demonstrates **exploratory data analysis (EDA)** using **Power BI** to help a retail organization move from reactive reporting to **predictive analytics**. Built-in **AI-powered features** are used to analyze trends, forecast future sales, detect anomalies, and generate automated insights.

Tools & Technologies

* **Power BI Desktop**
* CSV datasets (Sales & Forecast data)
* Power BI AI features (Forecasting, Anomaly Detection, Smart Narratives)

Key Features

* Historical sales trend analysis using line charts
* Seasonality analysis across product categories
* **AI-based sales forecasting** with confidence intervals
* **Anomaly detection** to identify unusual sales patterns
* **Smart Narratives** for automatic insight generation
* Comparison of actual vs forecasted sales

Business Insights

* Identifies peak demand periods and sales patterns
* Highlights abnormal sales behavior for risk analysis
* Supports data-driven decision making for inventory and planning

 Output

* Interactive Power BI report
* Exported PDF for academic submission
